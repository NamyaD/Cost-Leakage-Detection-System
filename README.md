Procurement Cost Leakage Analytics System
Overview

This project is a data-driven procurement analytics system designed to identify unnecessary, repetitive, and potentially wasteful business expenditures.

Using Python and Power BI, the system analyzes procurement transactions, detects abnormal spending patterns, classifies transaction risks, and provides interactive dashboards for business intelligence and cost optimization.

Features
-Duplicate transaction detection
-High-cost transaction analysis
-Rule-based leakage risk scoring
-Supplier-wise spending analysis
-Category-wise spending insights
-Interactive Power BI dashboard
-Risk classification of procurement transactions

Tech Stack
-Python
-Pandas
-NumPy
-Matplotlib
-Power BI
